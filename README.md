# ☕ Coffee Shop Landing Page

This project is a beautifully designed landing page for a modern cafe. It focuses on creating a "warm and cozy" atmosphere through careful color selection and professional layouts.

### 🚀 Key Features:
- **Hero Section:** Engaging visuals with call-to-action buttons for reservations.
- **Digital Menu:** A structured and stylish presentation of coffee and snacks.
- **Testimonials:** Clean layout for customer reviews and feedback.
- **Fully Responsive:** Optimized for a seamless experience on mobile and desktop.
- **Modern UI:** Using custom brown/cream color palettes and elegant typography.

### 🛠️ Technologies Used:
- **HTML5:** For semantic and organized structure.
- **CSS3:** For Flexbox/Grid layouts, custom styling, and responsive media queries.
